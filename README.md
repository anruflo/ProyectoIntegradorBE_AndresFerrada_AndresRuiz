# Proyecto Integrador Backend I

Este proyecto es un sistema de gestión para una clínica odontológica que permite crear, consultar, actualizar y eliminar datos de pacientes y odontólogos.

Se utilizaron tecnologías como:

-  Base de datos H2
-  ORM con Hibernate
-  Testeo Unitario con JUnit
-  Captura de excepciones
-  HQL/JPA
-  Logger

Implementación de capas y separación de lógica de negocio:

-  Entity
-  Controller
-  Service
-  Repository

Conexión e implementación básica de [frontend](https://github.com/and-fer/Clinica_Andreses_front).
